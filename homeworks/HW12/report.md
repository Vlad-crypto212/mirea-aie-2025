# Отчет по HW12: Временные ряды, лаговые признаки и GRU

## 1. Кратко: что сделано

В рамках данной работы были реализованы и протестированы 4 подхода к прогнозированию временного ряда:
- **B1**: Наивный прогноз (последнее значение)
- **B2**: Прогноз на основе скользящего среднего (7 дней)
- **B3**: Ridge регрессия на лаговых, скользящих и календарных признаках
- **R1**: GRU модель на оконном представлении (window=14)

Все эксперименты выполнены с корректной валидацией по времени и без утечки информации из будущего. Полные результаты всех экспериментов представлены в файле [`artifacts/runs.csv`](artifacts/runs.csv).

## 2. Среда и воспроизводимость

- Python 3.8+
- PyTorch 1.13+
- scikit-learn 1.2+
- pandas, numpy, matplotlib, seaborn
- Seed фиксирован: `987654321`
- Все эксперименты воспроизводимы благодаря фиксации генераторов случайных чисел и одинаковому временному сплиту.

## 3. Данные и постановка задачи

Использован датасет `S12-hw-dataset.csv` (4320 наблюдений):
- Колонки: `date` (datetime), `target` (числовой ряд)
- Диапазон дат: `2025-01-01 00:00:00` – `2025-06-29 23:00:00`
- Пропусков нет
- Цель: прогноз следующего значения ряда (horizon = 1)

## 4. Temporal split и признаки

Данные разделены по времени:
- **Train**: 2584 наблюдения (`2025-01-01` – `2025-04-18`)
- **Validation**: 862 наблюдения (`2025-04-19` – `2025-05-24`)
- **Test**: 862 наблюдения (`2025-05-25` – `2025-06-29`)

Разбиение визуализировано на графике [`artifacts/figures/series_split.png`](artifacts/figures/series_split.png).

Созданы следующие признаки (без утечки из будущего):
- Лаговые: `lag_1`, `lag_7`, `lag_14`
- Скользящие: `rolling_mean_7`, `rolling_std_7`
- Календарные: `day_of_week`, `month`, `day_of_year`

## 5. Модели и эксперименты (B1, B2, B3, R1)

| Эксперимент | Модель | Признаки | Основная особенность |
|-------------|--------|----------|----------------------|
| **B1** | Наивный (last value) | — | Простейший baseline |
| **B2** | Скользящее среднее (7 дней) | — | Усреднение последних 7 значений |
| **B3** | Ridge (α=1.0) | 8 лаговых/скользящих/календ. признаков | Линейная модель с L2-регуляризацией |
| **R1** | GRU (h=64, 2 слоя) | `target` только (окно=14) | Рекуррентная нейросеть |

## 6. Результаты

Сравнение по MAE на валидации показано на графике [`artifacts/figures/baselines_compare.png`](artifacts/figures/baselines_compare.png):

| Эксперимент | Val MAE | Val RMSE | Val MAPE | Test MAE |
|-------------|---------|----------|----------|----------|
| B1          | 24.9316 | 29.3263  | 0.1629   | 33.2030  |
| B2          | 34.8367 | 38.9293  | 0.2294   | 44.2810  |
| **B3**      | **5.8009** | **7.2406** | **0.0403** | **5.7562** |
| R1          | 7.9672  | 10.5036  | 0.0529   | 12.1778  |

Лучшая модель по валидационному MAE — **B3 (Ridge)**.

Кривые обучения GRU модели показаны на графике [`artifacts/figures/gru_learning_curves.png`](artifacts/figures/gru_learning_curves.png).

Прогноз лучшей модели (B3) на тестовых данных представлен на графике [`artifacts/figures/best_forecast_test.png`](artifacts/figures/best_forecast_test.png).

## 7. Анализ

- **B3 (Ridge)** превзошла все остальные модели, включая GRU, что говорит о высокой информативности engineered-признаков (лаги, скользящие статистики, календарь) для данного ряда.
- GRU показала худший результат на тесте (MAE = 12.18), вероятно, из-за переобучения или недостаточной глубины/гиперпараметров.
- Baseline методы (B1, B2) служат важной точкой отсчета — их MAE значительно выше, чем у B3.
- Все модели обучались с корректным временным сплитом, что гарантирует честную оценку.

## 8. Итоговый вывод

Все 4 эксперимента успешно проведены в соответствии с требованиями:
- Корректный temporal split (без random shuffle)
- Лаговые, rolling и календарные признаки построены без утечки
- GRU реализована через оконное представление
- Лучшая модель выбрана по validation, финальная оценка — на test (один раз)
- Все артефакты сохранены: `runs.csv`, `best_gru.pt`, `best_gru_config.json`, графики

Лучшая модель — **Ridge (B3)** с validation MAE = **5.8009**, test MAE = **5.7562**.  
Это подтверждает, что для многих временных рядов простые модели с хорошей инженерией признаков могут превосходить сложные нейросети.
