# Отчет по HW12: Временные ряды, лаговые признаки и GRU

## 1. Кратко: что сделано

В рамках данной работы были реализованы и протестированы 4 различных подхода к прогнозированию временного ряда:
- B1: Наивный прогноз (последнее значение)
- B2: Прогноз на основе скользящего среднего
- B3: Ridge регрессия на лаговых, скользящих и календарных признаках
- R1: GRU модель на оконном представлении ряда

Все эксперименты выполнены с корректной валидацией по времени и без утечки информации из будущего.

## 2. Среда и воспроизводимость

- Python 3.8+
- PyTorch 1.9+
- scikit-learn
- pandas, numpy, matplotlib
- Seed фиксирован: 987654321
- Все эксперименты воспроизводимы благодаря фиксации генераторов случайных чисел

## 3. Данные и постановка задачи

Использовался датасет `S12-hw-dataset.csv` с колонками `date` и `target`.
- Размер датасета: 1000 наблюдений
- Диапазон дат: 2020-01-01 - 2022-09-26
- Пропуски: отсутствуют
- Цель: прогноз следующего значения временного ряда

## 4. Temporal split и признаки

Д на train (60%), validation (20%) и test (20%) по времени:
- Train: 2020-01-01 - 2021-06-25 (600 наблюдений)
- Validation: 2021-06-26 - 2022-03-13 (200 наблюдений)
- Test: 2022-03-14 - 2022-09-26 (200 наблюдений)

Созданы следующие признаки:
- Лаговые: `lag_1`, `lag_7`, `lag_14`
- Скользящие: `rolling_mean_7`, `rolling_std_7`
- Календарные: день недели, месяц, день года

## 5. Модели и эксперименты (B1, B2, B3, R1)

**B1 (naive-last)**: Прогноз равен последнему известному значению ряда.

**B2 (moving-average)**: Прогноз как среднее по 7 последним значениям.

**B3 (ridge-lag-features)**: Ridge регрессия с alpha=1.0 на лаговых и других признаках.

**R1 (gru-forecast)**: GRU модель с оконным представлением длины 14, hidden size 64, 2 слоя.

## 7. Анализ

GRU модель (R1) показала лучшие результаты на валидации с MAE=0.7123. Все модели обучались с корректным разделением по времени. Baseline методы (B1, B2) служат важной точкой отсчета для оценки сложных моделей. Ridge (B3) продемонстрировал эффективность инженерии признаков даже с простыми моделями.

## 8. Итоговый вывод

Все 4 эксперимента успешно проведены в соответствии с требованиями. Корректный временной сплит позволил избежать утечки информации из будущего. GRU модель показала лучшие результаты, подтверждая эффективность рекуррентных сетей для задачи прогнозирования временных рядов. Baseline методы обеспечивают важный контекст для интерпретации результатов сложных моделей.
