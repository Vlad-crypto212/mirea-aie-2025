# Отчет по домашнему заданию HW10-11: Компьютерное зрение в PyTorch

## 1. Кратко: что сделано
Выполнены эксперименты по компьютерному зрению: задача классификации изображений (часть A) и задача детекции объектов (часть B). В части A реализованы 4 эксперимента с разными архитектурами и подходами. В части B выполнены 2 эксперимента с разными порогами уверенности для детекции.

## 2. Среда и воспроизводимость
- Python 3.8+
- PyTorch 1.9+
- Torchvision
- Все эксперименты выполнены с фиксированным seed=42 для воспроизводимости
- Использовано устройство: GPU (если доступно), иначе CPU

## 3. Данные
- Часть A: STL10 (10 классов, ~5000 обучающих изображений)
- Часть B: Pascal VOC (задача детекции объектов)
- Изображения приведены к размеру 224x224 для совместимости с ResNet
- Данные разделены на train/val/test с соотношением 80/20 для STL10

## 4. Часть A: модели и обучение (C1-C4)
- C1: Простая CNN без аугментаций (baseline)
- C2: Та же CNN с аугментациями (флип, поворот, цвет)
- C3: ResNet18 с предобученными весами, обучение только головы
- C4: ResNet18 с предобученными весами, частичный fine-tuning (layer4 + fc)

## 5. Часть B: постановка задачи и режимы оценки (V1-V2)
- Задача: детекция объектов на изображениях
- Использована предобученная модель Faster R-CNN
- V1: порог уверенности 0.3 (больше детекций, ниже точность)
- V2: порог уверенности 0.7 (выше точность, ниже полнота)

## 7. Анализ
- Transfer learning (C3, C4) значительно превосходит обучение с нуля (C1, C2)
- Аугментации данных (C2 vs C1) улучшают обобщающую способность
- Fine-tuning (C4) дает лучшие результаты чем обучение только головы (C3)
